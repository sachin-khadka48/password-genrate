# Password Generator

## Overview
The Password Generator is a simple web application that allows users to generate strong and secure passwords based on their preferences. Users can select the length of the password, choose to include uppercase letters, lowercase letters, numbers, and special characters. The application generates a random password that meets the specified criteria and provides an option to copy the password to the clipboard.

## Features
- Users can select the desired password length (e.g., 8, 12, 16 characters).
- Options to include uppercase letters, lowercase letters, numbers, and special characters.
- A button to generate a new random password.
- A display area for the generated password.
- A copy password button for easy copying to the clipboard.

## Technologies Used
- **HTML**: Structure of the webpage.
- **CSS**: Design and styling of the application.
- **JavaScript**: Logic for password generation and user interaction.

## Usage Instructions
1. Open `index.html` in a web browser.
2. Select the desired password length.
3. Choose the character types you want to include (uppercase, lowercase, numbers, special characters).
4. Click the "Generate" button to create a new password.
5. The generated password will be displayed on the screen.
6. Click the "Copy Password" button to copy the password to your clipboard.
